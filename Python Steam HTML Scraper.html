Python Steam HTML Scraper

Developed for my Post Graduate Research

Created for use in my post graduate research, this scraper once provided with a Steam app ID can scrape that applications discussion page, grabbing all posts and their associated meta data (thread, OP, Number of Replies, Date Posted, etc) and store it in a CSV file.

- Please Note: Whilst not violating Steam's terms of service, using a HTML scraper can be viewed as unusual traffic and may cause an IP ban, use at your own risk.

- Application may be out of date, due to the nature of the application it is not constantly updated, only updated when neccessary.
