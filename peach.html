Peach

"Peach" is a simple Mario clone developed in Unity 3D using C#. It was my first real attempt at making a game and taught me alot about programming as a whole.

Developed for a university assignment, Peach was produced in just under a month with all assets being created from scratch.

Whilst this game is very simple, I still view as the most aesthetically pleasing game I have produced so far. Sadly the project files were lost, all that is left of the game exists on Unity's webplayer now.
